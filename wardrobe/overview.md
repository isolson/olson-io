# Capsule Wardrobe v1

Minimal, durable wardrobe for Ely MN climate — outdoor work, travel, hiking, skiing, and casual-to-smart settings.

## Color Palette

| Role     | Colors                |
|----------|-----------------------|
| Neutrals | Navy, olive, charcoal, tan |
| Leather  | Brown / chestnut      |
| Accents  | Rust, burgundy        |

**Principle:** 3 colors max per outfit.

## System Summary

- **Footwear:** 8 pairs
- **Core clothing:** ~20 pieces
- **Accessories:** Essentials only
- **Activity gear:** Skiing + workout
- **Design goal:** Durable, mixable, minimal redundancy

## Closet Organization

| Wardrobe | Contents |
|----------|----------|
| 1 | Winter gear — insulation, base layers, ski clothing |
| 2 | Everyday clothing — pants, shirts, boots |
| 3 | Summer / BWCA gear — hiking gear, warm-weather items |
